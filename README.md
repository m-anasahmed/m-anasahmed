## 👋 Hey, I'm Anas

I never planned to become a developer. My plan was Mechanical Engineering — but an accidental CS enrollment in 2022 changed everything. Built my first project (a clothing catalog for a university lab), felt what it was like to make something exist that didn't exist before, and never looked back.

Today I build full-stack web applications, desktop software, and AI-powered platforms for real clients with real requirements. Based in Karachi — working locally and remotely.

## 🚀 Featured Projects

### 🎯 SmartFest — AI Event Venue Discovery Platform
> Final Year Project @ Sir Syed University of Engineering & Technology

Pakistan's first AI-powered event venue discovery platform. Three portals (User, Venue Owner, Admin), an AI Scheduler Agent, and an AI Budget Optimizer. Built with Next.js, Node.js, PostgreSQL, and integrated AI APIs.

---

### 🖥️ Klaar — Offline POS System
> Desktop Application for a local food business

Fully offline Electron desktop POS — receipt printing, inventory alerts, discount coupons, 8 report types. Replaced an entire paper-based system and is in daily use at their location.

`Electron` `React` `SQLite` `Node.js`

---

### 💎 Al-Sufael — Luxury Fragrance Dashboard
> Enterprise Admin Dashboard

Premium admin dashboard for a luxury fragrance brand. Product catalog management, order tracking, customer CRM — built to match their brand identity.

`React` `Node.js` `PostgreSQL` `Express`
