## 👋 Hey, I'm Anas

I never planned to become a developer. My plan was Mechanical Engineering — but an accidental CS enrollment in 2022 changed everything. Built my first project (a clothing catalog for a university lab), felt what it's like to make something exist that didn't before, and never looked back.

Today I build full-stack web applications, desktop software, and AI-powered platforms for real clients. Based in Karachi — working locally and remotely.
